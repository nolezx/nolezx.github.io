Инструкция по перепрошивке на Sailfish X

Для разблокировки загрузчика нам понадобятся:

    Xperia X

    Компьютер с Windows

    Fastboot драйвера

Этап 1. Загрузите двоичный образ с именем SW_binaries_for_Xperia_AOSP_M_MR1_3.10_v13_loire.zip для Xperia X с веб-сайта Sony.

Этап 2. Распакуйте zip архив.

Этап 3. Скопируйте извлеченный файл и вставьте его в тот же каталог, в который вы извлекли файл .zip Sailfish OS. После этого у вас должен быть SW_binaries_for_Xperia_AOSP_M_MR1_3.10_v13_loire.img в том же каталоге, что и fastboot.exe, flash-on-windows.bat, sailfish.img001 и некоторые другие файлы.

Этап 4. Загрузите Xperia в режим FastBoot.

Этап 5. Откройте файл "flash-on-windows.bat". Появится окно консоли.

Этап 6. Следуйте инструкциям в окне консоли.

Этап 7. Включите Xperia X.

Готово!
